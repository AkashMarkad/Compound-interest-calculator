# Compound-interest-calculator

Formula -

A = P(1 + \frac{r}{n})^{nt}

A	=	final amount.
P	=	initial principal balance.
r	=	interest rate.
n	=	number of times interest applied per time period.
t	=	number of time periods elapsed.
